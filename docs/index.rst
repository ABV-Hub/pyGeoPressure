Overview
--------

*Tools for pore pressure prediction using well log data and seismic velocity data.*

Features
--------

1. Hydrostatic Pressure Calculation
2. Overburden (or Lithostatic) Pressure Calculation
3. Eaton's method
4. Bowers' method
